# About nuxeo-document-permissions

**Nuxeo Document Permissions** is an element for managing document permissions in Nuxeo.

## Dependencies

Element dependencies are managed via [Bower](http://bower.io/). You can
install that via:

    npm install -g bower

Then, go ahead and download the element's dependencies:

    bower install

## Quickstart

We recommend that you use [Polyserve](https://github.com/PolymerLabs/polyserve) which you can install via:

    npm install -g polyserve

And you can run it via:

    polyserve -p 3000

Once running, you can checkout the docs and demo at `http://localhost:3000/components/nuxeo-document-permissions/`.
